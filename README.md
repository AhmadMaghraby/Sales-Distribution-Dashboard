# Sales Distribution Dashboard

## Project Overview

This project presents an interactive **Sales Distribution Dashboard** developed using **Microsoft Excel** to analyze sales performance across multiple business dimensions. The dashboard transforms raw sales data into meaningful insights, enabling stakeholders to monitor performance, identify trends, and support data-driven decision-making.

## Objectives

- Analyze sales performance across regions, customers, and products.
- Compare actual sales against business targets.
- Monitor sales trends over time.
- Identify top-performing customers and products.
- Support strategic business decisions through interactive reporting.

## Dataset

The dataset contains transactional sales records with the following attributes:

- Date
- Customer Name
- Product
- Unit Price
- Quantity
- Actual Sales
- Country
- Region
- Month
- Week

## Dashboard Features

- Interactive Sales Dashboard
- Dynamic Filtering
- Regional Sales Analysis
- Customer Performance Analysis
- Product Performance Analysis
- Target vs. Actual Sales Comparison
- Time-Series Sales Analysis
- Business KPI Monitoring

## Tools & Technologies

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Power Query
- Data Cleaning
- Data Analysis
- Data Visualization

## Business Value

The dashboard enables decision-makers to monitor sales performance from multiple perspectives, evaluate business targets, identify high-performing regions and customers, and uncover sales opportunities through interactive visualizations and analytical reporting.
